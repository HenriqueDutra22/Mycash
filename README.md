
# 💎 MyCash - Controle Financeiro Inteligente

![Banner](https://images.unsplash.com/photo-1554224155-6726b3ff858f?auto=format&fit=crop&q=80&w=1000&h=300)

O **MyCash** é uma plataforma moderna e intuitiva projetada para transformar a maneira como você lida com seu dinheiro. Combinando um design premium com funcionalidades avançadas, o MyCash oferece controle total sobre suas receitas, despesas e metas financeiras.

## ✨ Funcionalidades Premium

- 🌓 **Design Ultra Moderno**: Interface construída com foco em sofisticação, utilizando Glassmorphism e animações fluidas.
- 🕵️ **Modo Fantasma (Privacidade)**: Esconda seus saldos e gráficos com um clique para usar o app em locais públicos.
- 📸 **Scanner de Notas (OCR)**: Adicione transações apenas tirando uma foto da nota fiscal através de leitura inteligente.
- 📊 **Análises Avançadas**: Gráficos interativos para visualizar fluxo de caixa mensal, gastos por categoria e métodos de pagamento.
- 💳 **Gestão de Cartões**: Personalize seus métodos de pagamento (Nubank, Inter, Visa, etc) e acompanhe os gastos de cada um.
- 🔄 **Recorrência e Parcelas**: Gerencie despesas fixas mensais e acompanhe o progresso de compras parceladas.
- 🎯 **Meta Mensal**: Defina um limite de gastos e acompanhe através de uma barra de progresso inteligente no dashboard.
- 🎨 **Personalização Total**: Escolha a cor de destaque (Accent Color) que mais combina com seu estilo.

## 🛠️ Tecnologias Utilizadas

- **Frontend**: [React.js](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
- **Banco de Dados & Autenticação**: [Supabase](https://supabase.com/)
- **Gráficos**: [Recharts](https://recharts.org/)
- **Processamento de Imagem**: [Tesseract.js](https://tesseract.projectnaptha.com/)
- **Ícones**: [Google Material Symbols](https://fonts.google.com/icons)

## 🚀 Como Rodar o Projeto

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/HenriqueDutra22/Mycash.git
   ```

2. **Instale as dependências**:
   ```bash
   npm install
   ```

3. **Configure o Supabase**:
   Crie um arquivo `.env` ou configure as credenciais no `supabaseClient.ts`:
   - `supabaseUrl`
   - `supabaseAnonKey`

4. **Inicie o servidor de desenvolvimento**:
   ```bash
   npm run dev
   ```
